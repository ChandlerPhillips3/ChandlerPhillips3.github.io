March Madness Simulation

  • Utilized Python and each team's data in order to help generate my bracket for the NCAA Men’s Division 1 basketball tournament
  • Factoring in each team’s statistics, I programmed a simulation in order to determine the probabilistic outcome of each basketball game match-up
  • With the help of the Numpy library I used a ratio of each teams probability to win against their opponent, as well as the random module to help decide who would move on based on their probability, all while using the matplot library in order to visualize these results
  • My simulated bracket took first place out of twenty five entries

Data structures Time complexity project

  • Used C++ in order to implement two different data structures and two different sorting algorithms: A Doubly Linked List, a Hash Table with Quadratic Probing, a Bubble Sort Algorithm, and a Heap Sort Algorithm
  • For each data structure and algorithm I measured and stored in nanoseconds the time it took to insert 100 elements, as well as the time it took to search for 100 random elements already inserted
  • After inserting 10,000 different elements, I plotted these measured times for each data structure and algorithm using excel, and was able to compare and contrast each data structure and algorithm's Big-O time complexity for insertions and searches by actually visualizing how the time complexity can change depending on how many items are already being stored
